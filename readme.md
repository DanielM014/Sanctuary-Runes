# Sanctuary Runes

Juego 2D top-down de supervivencia arena. Eres un aprendiz de mago atrapado en un santuario en ruinas: apunta con el ratón y dispara con clic izquierdo para sobrevivir oleadas de espíritus el mayor tiempo posible.

# Ciclo jugable

Arena cerrada.

Enemigos aparecen de forma progresiva y persiguen al jugador.

El jugador dispara proyectiles magicos con cooldown.

Si la vida llega a 0 se acaba la partida.

Se guarda un record (mejor partida).


# Controles v0.1

WASD: movimiento

Mouse: apuntar

Clic izquierdo: disparar

Esc: pausar / volver al menú

# Puntaje y records
Métrica principal (v0.1): tiempo sobrevivido (segundos).
Métrica secundaria (v0.1): eliminaciones (kills)

- Leaderboard local: Top 10 puntajes en archivo

# Alcance del Proyecto

Incluye en v0.1 (MVP jugable)

- Movimiento top-down del jugador.

- Disparo manual: apunta con mouse + clic izquierdo.

- Proyectiles con colisiones y daño.

- 1 enemigo basico (IA simple: perseguir).

- Spawner con dificultad creciente (aumenta frecuencia con el tiempo).

- Vida/daño + Game Over + reinicio.

- UI minima: vida, tiempo, kills.

- Guardado/carga de Top 10 local.

- Menu principal y pantalla de Game Over.
